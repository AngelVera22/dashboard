# Capacitación JavaScript

## Unidad 1: Introducción
Conceptos básicos de JavaScript, variables y uso de la consola.

## Unidad 2: Strings
Manipulación de cadenas de texto, métodos, substrings y template strings.

## Unidad 3: Numbers
Operaciones numéricas, conversiones de tipos, parseInt y funciones matemáticas.

## Unidad 4: Variables
Uso de let y const, buenas prácticas para la declaración de variables.

## Unidad 5: Conditions
Estructuras condicionales, operadores de comparación y operador ternario.
