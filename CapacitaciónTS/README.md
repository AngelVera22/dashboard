# Capacitación TypeScript

## Unidad 2: Configuración y Seguridad de Tipos

Uso del tipo `any`, configuración mediante `tsconfig.json` y aplicación del modo estricto (`strict`) para mejorar la detección de errores.

## Unidad 3: TypeScript ESLint

Herramientas de análisis estático, reglas de calidad de código, restricciones sobre `any` y buenas prácticas para el manejo de errores de tipos.

## Unidad 4: Tipos Primitivos e Inferencia de Tipos

Tipos primitivos de JavaScript y TypeScript, anotaciones de tipos e inferencia automática para reducir código repetitivo.

## Unidad 5: Union Types y Narrowing

Creación de tipos que aceptan múltiples valores, uso de Union Types y técnicas de Narrowing para determinar tipos específicos de forma segura.

## Unidad 6: Type Aliases

Definición de alias de tipos para mejorar la reutilización, legibilidad y mantenimiento del código TypeScript.
